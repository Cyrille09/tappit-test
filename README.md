# Tappit Task

## Installation and starting process

```bash
### Clone the repo

git clone https://github.com/Cyrille09/tappit-test.git

### redirect to the task folder the below command

cd tappit-test

### Install the dependencies

npm install

# run the task

npm start

# run the test

npm test

```
